      # UTAU 引擎调用

      最重要的后端处理过程————引擎调用。

      # UTAU 合成声音原理

      • 将人工录制的声音样本 拉伸、变调（重采样），再拼接（合成）。

      # 引擎

      • 引擎分为合成器与重采样器，重采样器进行拉伸变调工作，合成器对重采样器的输出文件进行拼接。
      • 合成后的音频质量由音源本身质量决定，但很大程度上受重采样器的算法影响。
      • 引擎都是控制台程序，没有 GUI 框架，运行时在控制台输出信息供用户参考，执行完毕立即返回。
